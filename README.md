> **A cutting-edge deep learning framework for robust Channel State Information (CSI) compression and temporal prediction in extreme high-mobility scenarios (up to 120 km/h).**

## 📖 1. Project Overview
In 5G-Advanced and 6G FDD Massive MIMO systems, accurately obtaining Downlink CSI is the bottleneck for optimal beamforming. Traditional non-AI methods suffer from severe **"Outdated CSI"** issues under high Doppler shifts (e.g., high-speed trains) and consume massive uplink feedback bandwidth. 

**iCsiNet-2D** solves these challenges by introducing an AI-Native air-interface design. It dramatically compresses the CSI payload to a strict 4-bit limitation while predicting the multi-path fading evolution, achieving near-perfect beam alignment even at 120 km/h.

## ✨ 2. Core Highlights
- 🧊 **Ultra-Lightweight UE via 2D Perception**: Replaces traditional 1D convolutions with **Depthwise Separable Conv2d**, extracting spatial-spectral coupled features while reducing mobile terminal (UE) FLOPs by >80%.
- ⏳ **Temporal Inversion with iTransformer**: Abandons standard LSTMs. Implements the state-of-the-art `iTransformer` at the Base Station (BS), treating spatial features as independent tokens to perfectly capture long-range Doppler time-variance.
- 🗜️ **Adaptive 4-bit LSQ Compression**: Fully complies with 3GPP strict payload constraints. Introduces **Learned Step Size Quantization (LSQ)**, limiting the entire feedback payload to just 1024 bits without sacrificing structural integrity.
- 🎯 **Physics-Informed Beam Alignment**: Proposes a hybrid loss function combining Smooth-MSE (differential penalty) and **SGCS (Generalized Cosine Similarity)**. Forces the neural network to prioritize physical beam phase alignment over pure magnitude reconstruction.

## 📊 3. Breakthrough Performance (OOD Generalization)
